# Tech Stack

## Programming Language
- **Python 3.10+**

## UI Framework
- **Tkinter**: Python 标准 GUI 库，用于创建原生 Windows 界面。

## Video & Image Processing
- **FFmpeg**: 核心处理引擎，用于无损剪辑视频（Stream Copy）和提取元数据。
- **OpenCV (opencv-python)**: 用于快速提取视频帧并生成预览图。
- **Pillow (PIL)**: 用于将 OpenCV 提取的帧转换为 Tkinter 可显示的格式。

## Packaging
- **PyInstaller**: 用于将项目打包成独立的 Windows `.exe` 文件。

## Development Tools
- **pip**: 包管理器。
- **venv**: Python 虚拟环境。
