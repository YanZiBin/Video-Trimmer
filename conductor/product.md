# Product Definition

## Vision
一款轻量级的桌面端工具，专为优化 AI 图生视频工作流而设计。它解决了生成视频末尾帧质量不佳，导致无法作为下一段视频参考图的痛点，通过精准剪除尾部帧，节省重新生成的时间和算力。

## Core Functionality
1. **视频导入**: 支持主流视频格式导入。
2. **精准剪辑**:
   - 提供**数字输入框**，用户直接输入需要去掉的末尾帧数。
3. **即时预览**:
   - 界面实时显示处理后的**最后一帧静态图片**。
   - **不包含**视频播放功能，专注于确认末尾帧画面。
4. **无损输出**:
   - 剪辑后的视频严格保持原视频的格式、编码、码率等所有参数。
   - 采用**手动选择路径**的方式保存文件，由用户指定文件名和位置。

## User Experience
- **界面**: 简洁的 Windows 桌面 GUI。
- **流程**: 导入视频 -> 输入数字 -> 查看预览图 -> 满意保存 -> 不满意修改数字。

## Deployment
- **交付形式**: 封装为可直接运行的 `.exe` 文件（Windows）。
